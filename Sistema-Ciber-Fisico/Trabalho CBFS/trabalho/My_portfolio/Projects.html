Vending Machine

- Raciocínio algorítimo

O projeto é um programa que simula uma máquina de vendas. Ele possui duas opções de usuário: administrador e consumidor. O programa permite que o administrador atualize o troco na máquina, atualize o estoque de produtos e verifique a quantidade de dinheiro das vendas. Para o consumidor, o programa permite selecionar um produto para comprar, inserir o dinheiro correspondente e receber o troco, se houver.

No geral, o código simula o funcionamento básico de uma máquina de vendas, permitindo que os usuários realizem compras e que o administrador gerencie o estoque e as finanças da máquina.
